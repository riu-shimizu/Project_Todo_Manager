# Project Manage Todo — Glassmorphism Wireframe

## トーン&ガイド
- 背景: パステル系のグラデ＋ソフトなノイズ、カードはガラス質 (半透明/白ボーダー/blur)。
- アクセント: ブルー (#3b82f6) とオレンジ (#f97316) をワンポイント、角は 12px 前後のラウンド。
- タイポ: Inter + Noto Sans JP、ウェイトは 500/600 を基調。

## ページ共通
- 左サイドバー: プロジェクト一覧 + Today への切替、半透明ダークカード。
- メイン: 上にヘッダー (プロジェクト名/説明と進捗サマリ)、その下に幅いっぱいのガントチャート、さらに下に階層編集ボード。
- カード: 透明度 70–80%、白ボーダー、ドロップシャドウ、blur(12–16px)。

```
┌──────── Sidebar ───────┬──────────────────── Main ────────────────────┐
│ Projects / Today switch │ Header (title, description, stats)          │
│                         │ Gantt (full width)                          │
│                         │ Hierarchy board (phases > works > tasks)    │
└─────────────────────────┴─────────────────────────────────────────────┘
```

## プロジェクト画面
1. ヘッダー: プロジェクト名・説明、4段の進捗バッジ（大/中/小項目/Todo）。
2. ガントチャート: 横スクロール、Today ライン、ステータス色 (未着手=グレー/進行中=シアン/完了=グリーン)。
3. 階層ボード: 各行はタイトル + 予定/実績日入力 + 進捗バー + 削除ボタン。ステータスセレクトは無し（実績日で自動判定）。

## Today 画面
- 全プロジェクトの「今日の Todo」を 1 リストで表示（プロジェクト名タグ付き）。
- ステータス変更/タイトル編集のみのシンプル行。背景は薄いガラスカード。

## 操作のポイント
- ステータスは実績日から自動判定。入力フィールドはタイトル・予定/実績日・メモに限定。
- 追加/削除ボタンはゴーストスタイルでミニマルに。
- ホバーでカードのグローと軽い浮き上がりを付与。
